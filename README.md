***НАЧАЛО РАБОТЫ:***

Чтобы запустить проект локально:

1. Клонировать репозиторий
   ```console
   git clone https://github.com/NastyaNev/test-posts.git
   ```

2. Установить зависимости из package.json
   ```console
   npm install
   ```
   
3. Запустить проект локально
   ```console
   npm start
   ```
